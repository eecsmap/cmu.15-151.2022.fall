
### some useful symbols

* $a \impliedby b$
  
  read $\impliedby$ as **implied by** or **if**
  
  if $b$ then $a$
  
  $a$ if $b$

* $a \iff b$
  
    read $\iff$ as **if and only if**
    
    $a$ if and only if $b$
    
    $a$ if $b$ and $b$ if $a$

* $a \circ b$
  
    read $\circ$ as **composed with**
    
    $a$ composed with $b$
    
    $a$ after $b$
